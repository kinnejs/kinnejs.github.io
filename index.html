Portfolio

Take Action!!!

Take Action!!! is an ASP.NET Web MVC Application that is used to keep track of Assignments for all of your Business Needs. This website was built with the help of Bootstrap and was created as a project for Eleven Fifty Academy.

Boats-4-U

Boats-4-U is an ASP.NET Web Application (.NET Framework) in the C# programming language. The purpose of this application is to create a platform in which would-be boat enthusiasts can safely connect with boat owners/operators for the purpose of water recreation.

Eleven Note MVC

Eleven Note is an ASP.NET Web Application (.NET Framework) in the C# programming language. The purpose of this application is to allow users to create Notes online. Other users can then add comments to the individual notes.

Gold Challenges

The Gold Challenges are a series of four Console Applications written in C#. The first application allows a manager at the Komodo Cafe to create, read update and delete items on the Cafe's menu. The second application allows a claims agent at Komodo Insurance to see all of the claims, work on the next item in the claim, and enter a new claim. The third application allows a security officer to assign badges access to certain doors. The final application allows a company to track outings as well as calculate the costs for the various outings.

Character

The Character is a project that was created with the sole use of HTML and CSS.


About Me

My Name is Jonathan Kinne. In 2009, I graduated from Vanderbilt University with a BA in Chemistry and Finance. After working two years as a Clinical Pharmacologist at Vanderbilt Medical Center I went back to school and earned an MBA from Belmont University with a concentration in Healthcare and Finance. After graduation I had held positions as a Financial and Business Analyst in Nashville, TN; Dallas, TX and Indianapolis, IN. While I have thoroughly enjoyed my time working as an analyst, I decided that it was time to make a change.
On May 13th 2021, I will graduate from ElevenFifty academy where I have been honing my skills as a Software Developer. In my time with ElevenFifty, my view of the software industry has definitely expanded. The enjoyment that I get from challenging myself along with being able to create projects from scratch have allowed me to understand that this is definitely the career path that I was meant to pursue.
